Como jugar


Jugar es la forma de pasar el tiempo libre con ayuda de algún programa de uso multiple.Realizar una actividad o hacer una cosa, generalmente ejercitando alguna capacidad o destreza, con el fin de divertirse o entretenerse.

Como trayhardear


Este termino se utiliza para definir el proceso por el cual un jugador o gamer destaca en un videojuego de forma que siempre intentara ganar. Para lograr ser "Tryhard" se necesita practicar mucho en un videojuego y conocer varias tacticas que ayuden en el desempeño del mismo haciendo que puedas ser el mejor
